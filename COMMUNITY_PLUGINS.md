# Community Plugins

Third-party Health Checker plugins created by the community.

> Want to add your plugin? Submit a PR to this file or [open an issue](https://github.com/mySites-guru/HealthCheckerForJoomla/issues).

| Plugin | Author | Description | Link |
|--------|--------|-------------|------|
| YOOtheme Pro Health Checker | Alex Revo | YOOtheme Pro Health Checker is a specialized diagnostic tool for websites built with YOOtheme Pro Builder. It automatically verifies server requirements, checks Joomla configuration settings, and reviews YOOtheme Pro setup - identifying potential issues and providing actionable recommendations to resolve them. This helps ensure stable operation, high performance, and adherence to development best practices. | https://alexrevo.pw/product/revo-debug-plugin.html |
