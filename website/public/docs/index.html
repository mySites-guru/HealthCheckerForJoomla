What is Health Checker? ‚Äã

Health Checker for Joomla is a comprehensive monitoring extension that checks your Joomla site's health across multiple categories. Think of it as a "systems check" that identifies potential problems before they become critical issues.

Quick Overview ‚Äã

  • 133 built-in checks across 8+ categories
  • Dashboard Widget showing Critical/Warning/Good counts at a glance
  • Detailed Reports with filtering by status and category
  • Export Functionality in JSON and HTML formats
  • Extensible - third-party developers can add their own checks
  • Super Admin Only - secure access control
  • No Persistence - fresh results on every run

For Site Administrators ‚Äã

Health Checker helps you:

  • Identify security vulnerabilities before they're exploited
  • Catch configuration issues early
  • Monitor database health
  • Ensure optimal performance settings
  • Verify SEO best practices
  • Maintain content quality

For Extension Developers ‚Äã

Health Checker provides a simple, event-based API for adding custom health checks to Joomla sites. Perfect for:

  • Monitoring your extension's configuration
  • Verifying required resources are available
  • Checking integration status with third-party services
  • Ensuring optimal settings for your extension
